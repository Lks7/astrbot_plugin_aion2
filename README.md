# astrbot_plugin_aion_tracker

永恒之塔多角色养成 AstrBot 插件。

## 功能

- 多角色绑定与切换
- 体力、噩梦券、讨伐券、觉醒券追踪
- 超越次数、远征次数、挑战次数追踪
- 自动恢复/重置结算
- 自然语言录入

## 指令

- `/at addchar 角色名 职业 [等级]`
- `/at switch 角色名`
- `/at status`
- `/at all`
- `/at allres`
- `/at add 资源名 数量`
- `/at update 体力 噩梦 讨伐 觉醒 超越 远征 挑战 基纳`
- `/at run 角色名 远征次数 超越次数`
